# Public Product Price Data Extraction (Excel Dataset)

## Project Overview
This project demonstrates how to extract public product price data from online product listing websites using Octoparse, a no-code web scraping tool.
The extracted data is cleaned, standardized, and saved in Excel (.xlsx) format for analysis and reporting.

## Tools & Technologies
- Microsoft Excel (.xlsx)
- Public product listing websites

## Project Objectives
- Extract product price data from public web pages
- Store data in Excel format for analysis

## Data Description
- Price: Cleaned numeric product prices

## Workflow
1. Select public product listing website
2. Extract prices using Octoparse
3. Clean data (remove symbols, empty values)
4. Export to Excel (.xlsx)
